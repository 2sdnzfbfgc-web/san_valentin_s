# Página "¿Quieres ser mi San Valentín?"

Abre `index.html` en tu navegador para usar la página interactiva.

- Archivos creados/actualizados:
  - `index.html` — estructura y botones
  - `styles.css` — estilos
  - `script.js` — lógica (vidas, respuestas)

Cómo abrir (local):

1. En VS Code, haz doble clic en `index.html` y usa la opción "Open in Default Browser" o
2. Desde la terminal, ejecuta:

```bash
# sirve para sistemas con xdg-open (Linux)
xdg-open index.html
```

Si quieres, añado confetti, animaciones extra o guardo el estado en localStorage.
